Deploying Certer is straightforward since it is built from the ground up to be container-native. Select one of the installation paths below to get started:

1. [Docker Run](docker/) - Quick execution on a single host.
2. [Docker Compose](docker-compose/) - Declarative local stack using compose files.
3. [Kubernetes](kubernetes/) - Scalable deployment in a cloud cluster.
