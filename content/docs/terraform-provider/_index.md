The Certer Terraform provider maps configuration to your Certer deployment, allowing teams to request certificates, provision client API keys, and scope permissions using HCL configurations.

1. [Provider Setup](setup/) - Install and configure the provider.
2. [Resources Reference](resources/) - Manage teams, certificates, and keys.
3. [Data Sources Reference](data-sources/) - Fetch certificate bodies and parse validity metadata.
